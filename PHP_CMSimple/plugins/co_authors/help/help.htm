Kurz-Dokumentation Plugin "CoAuthors"

Version CoAuthors 2.0 - released 05/2016

CoAuthors ist ein CMSimple Plugin, mit dem txt-Dateien, die html Quelltext enthalten und die irgendwo in einem Unterverzeichnis einer CMSimple Installation liegen, in eine Seite dieser CMSimple Installation eingebunden werden können.

Der Plugin Aufruf für CMSimplCoAuthors (4.1) entfernt alle h1 Überschriften aus dem Quelltext.

Mit dem CMS auf der Basis von CMSimple

CMSimpleCoAuthors »

können Sie diese txt-Dateien einfach und komfortabel
in Ihrer gewohnten Arbeitsumgebung erzeugen.

1. Systemvoraussetzungen

Voraussetzung ist eine funktionierende Installation von CMSimple 4.0 oder höher.

2. Installation

  1. zip-Datei herunterladen und mit einem geeigneten Programm entpacken
  2. Laden Sie den Ordner "co_authors" in das Verzeichnis "plugins" Ihrer CMSimple Installation hoch.

3. Datei- und Ordnerrechte

Die Vergabe von Schreibrechten für Ordner und Dateien ist nicht notwendig.

4. Plugin-Aufruf

4.1 Plugin-Aufruf für CMSimpleCoAuthors

Fügen Sie folgenden Plugin-Aufruf in Ihre CMSimple Seite ein, an der Stelle, an der die txt-Datei eingebunden werden soll:

{{{plugin:co_authors('CoAuthorFolder/','Die_Autoren_Seite');}}}

Hinweis: Bitte fügen Sie den Plugin Aufruf immer in der Quelltext Ansicht ein.

CoAuthorFolder: Das Verzeichnis, in dem CMSimpleCoAuthors - also die Autoren Installation - installiert ist.

!!! WICHTIG !!! der Slash ("/") am Ende, ohne den funktioniert's nicht.

Die_Autoren_Seite: Der Name der Seite der Autoren Installation, die eingebunden werden soll, mit "_" für die Leerstellen.

!!! WICHTIG !!! Bitte bei den Namen der Autorenseiten keine Umlaute oder Sonderzeichen verwenden, da daraus Dateinamen gemacht werden, die an CMSimple übergeben werden.

4.2 Plugin-Aufruf mit vollem URL

Fügen Sie folgenden Plugin-Aufruf in Ihre CMSimple Seite ein, an der Stelle, an der die txt-Datei eingebunden werden soll:

{{{plugin:co_authorsURL('Datei mit vollem URL');}}}

Hinweis: Bitte fügen Sie den Plugin Aufruf immer in der Quelltext Ansicht ein.

Beispiel - Example:

{{{plugin:co_authorsURL('./userfiles/texte/text01.txt');}}}

5. Konfiguration

Für CoAuthors gibt es keine Konfigurations Optionen.
